Set up your L2 worker

You run a small helper (a "worker") on your own computer that does the L2 work. You only set this up once — it takes a few minutes.

This saves setup-l2-worker.ps1 into your Downloads folder.

3. Run it in PowerShell
  1. Press the Windows key (bottom-left ⊞), type PowerShell, and press Enter. A blue window opens.
  2. Copy the line below, click inside the blue window, paste it with Ctrl+V (or right-click), and press Enter:
    powershell -ExecutionPolicy Bypass -File "$env:USERPROFILE\Downloads\setup-l2-worker.ps1"
  3. Let it run — it downloads and installs everything (a few minutes). If Windows asks permission to install something, click Yes/Allow.
  4. The first time, a browser window opens to the L2 login page — log in once. It remembers you after that.
  5. Leave the blue PowerShell window open — that is your worker. If you close it, your builds just wait at "pending" until you open it again.
  6. Come back to this page, set You: (top-right) to your email, and build an audience. Your worker picks it up.

On a Mac? Use .env only, then run the same steps in Terminal by hand: git clone the repo, npm install, npx playwright install chromium, drop the .env in, npm run build, npm run worker.
